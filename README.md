# ParallellisedSorter
A small experiment to make a quicksort algorithm that sorts an array in parallel.

# Some Results
## "I think they're called benchmarks"

|--------------------|------------|
| 10                 | 7ms        |
| 100                | 7ms        |
| 10,000             | 222ms      |
| 100,000            | 6s 468ms   |
